## Description

A thin wrapper around aws-sdk's v2 DynamoDB Client

Persist your objects to DynamoDB

requires `gem 'aws-sdk', '~>2'`

## Features
  - Partition/Sort Key Schema (including RANGE)
  - Local & Global Secondary Index Support

## Examples

    require 'dynamour'
